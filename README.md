# Crypto Project Tracker Frontend

This is the front end for the Crypto Tracker project. It is a Next.js application that uses MUI, a custom backend found [here](https://github.com/zarnoevic/crypto-tracker-be), and web3 non-custodial wallets for authentication.

# Running the app

1. Run the [Backend](https://github.com/zarnoevic/crypto-tracker-be)
2. Install dependencies

```bash
$ npm install
```

3. Run the frontend
    
```bash
$ npm run dev
```
